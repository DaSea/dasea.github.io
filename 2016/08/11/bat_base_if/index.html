if errorlevel语句特点:

  • if errorlevel 0 cmmand, 含义是:如果返回的错误码值大于或等于0的时,将执行cmmand操作
  • if %errorlevel%==0 cmmand，含义是:如果返回的错误码值等于0的时候,将执行cmmand操作
